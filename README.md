# StudyFlow – Organizador de Estudos Inteligente

## Descrição

O **StudyFlow** é uma aplicação web frontend desenvolvida com **HTML, CSS e JavaScript puro**, criada para ajudar estudantes a organizar tarefas escolares, acompanhar prazos e identificar automaticamente quais atividades devem ser priorizadas.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript Vanilla
- LocalStorage

## Funcionalidades

- Cadastro de tarefas com:
  - Nome da tarefa
  - Matéria
  - Data de entrega
  - Prioridade
- Exibição de tarefas em cartões modernos
- Botões para concluir e excluir tarefas
- Filtros por:
  - Todas
  - Pendentes
  - Concluídas
- Barra de progresso com porcentagem
- Contador total de tarefas
- Meta semanal de estudos
- Sistema inteligente de priorização
- Tema escuro com persistência no LocalStorage
- Validação de formulário e mensagens amigáveis de erro
- Interface responsiva e mobile first

## Como executar localmente

1. Baixe ou clone o projeto.
2. Abra o arquivo `index.html` em um navegador moderno.
3. Comece a cadastrar suas tarefas.
